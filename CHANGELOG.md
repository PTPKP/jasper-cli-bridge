# Changelog

All notable changes to `jasper-cli-bridge` will be documented in this file.

## [1.0.0] - 2026-03-02

### Added
- Initial release of JasperReports CLI Bridge
- PHP wrapper for JasperReports with Java CLI integration
- Support for JRXML template compilation and PDF generation
- JSON data source support
- Database connection support
- Laravel service provider and facade
- Configuration management
- Comprehensive documentation and examples
- MIT License

### Features
- Generate PDF reports from JRXML templates
- Support for report parameters
- Template existence checking
- Template listing functionality
- Java version detection
- Configurable JVM options
- Customizable output paths
- Laravel auto-discovery support
- PSR-4 autoloading

### Documentation
- Comprehensive README with examples
- Installation guide (INSTALL.md)
- Laravel controller examples
- Plain PHP usage examples
- Build script for Java CLI
